# TaskLoop

A modern Todo List web app built using HTML, CSS, and JavaScript.
TaskLoop helps you organize, track, and manage your daily tasks efficiently.

## Features
- Add new tasks quickly
- Delete tasks when completed
- Press Enter to add tasks
- Responsive and user-friendly design

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run Locally
1. Clone this repository:
   https://github.com/shravani27-07/TaskLoop.git
2. Open `index.html` in your browser
